## 실제 작성 예시

### 제목
```
2025년 봄 학기 개강 안내
```

### URL 슬러그
```
2025-spring-semester
```
(Generate 버튼으로 자동 생성)

### 내용

```markdown
안녕하세요, 핀토스입니다.

2025년 봄 학기가 다가왔습니다. 주요 일정을 안내드립니다.

## 주요 일정

- **개강일**: 3월 2일 (월)
- **중간고사**: 4월 20일 ~ 26일
- **기말고사**: 6월 15일 ~ 21일

[이미지: 2025년 학사일정 포스터]

자세한 내용은 [학사 안내 페이지](https://example.com)를 참고해주세요.

감사합니다.
```

### 이미지 삽입 방법
1. 본문에서 📷 아이콘 클릭
2. 파일 선택 또는 드래그 앤 드롭
3. 대체 텍스트: "2025년 학사일정 포스터"
4. 캡션: "봄 학기 주요 일정"

## 결과 확인

작성 후 다음 페이지에서 확인 가능:

1. **메인 페이지**: `http://localhost:3000`
   - 최신 5개 공지사항 표시됨

2. **공지사항 목록**: `http://localhost:3000/notice`
   - 전체 공지사항 목록 (페이지네이션)

3. **공지사항 상세**: `http://localhost:3000/notice/2025-spring-semester`
   - 작성한 내용 전체 표시

## FAQ 작성 방법

FAQ는 더 간단합니다:

1. 좌측 메뉴 "자주 묻는 질문" 클릭
2. "+" 버튼
3. **질문** 입력: "환불은 어떻게 하나요?"
4. **답변** 입력: "마이페이지 > 주문내역에서 환불 신청이 가능합니다."
5. **카테고리** 선택: 결제 관련
6. **순서** 입력: 1 (낮을수록 위에 표시)
7. Publish

---

## 수정/삭제

### 수정
1. 좌측 목록에서 항목 클릭
2. 내용 수정
3. "Publish" 클릭

### 삭제
1. 항목 열기
2. 우측 상단 "..." (더보기) 클릭
3. "Delete" 선택
4. 확인

---

## 비공개 설정 (임시저장)

"공개 여부" 체크 해제 후 Publish하면:
- Studio에서는 "[비공개]" 태그와 함께 보임
- 실제 사이트에서는 보이지 않음
- 나중에 체크 후 다시 Publish하면 공개됨
